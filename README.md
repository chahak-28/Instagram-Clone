This project is a Mini Instagram-style application created as part of an interview selection task.
I have mainly focused on building a strong backend using Spring Boot, with basic frontend screens to interact with the APIs.

**The goal of this project is to demonstrate:**

Backend logic

REST API design

Database relationships

Authentication handling

**Tech Stack**

Java 17

Spring Boot

Spring Security (JWT authentication)

Spring Data JPA

MySQL

Docker

MailHog (for local email testing)

React (basic frontend)

**Features Implemented**

User signup and login

Secure password hashing

JWT-based authentication

Create posts with image URL and caption

Like and comment on posts

Basic frontend screens for testing APIs

**Running the Project**
Using Docker
docker-compose up --build

**Services**

Backend API: http://localhost:8080

MySQL: localhost:3306

MailHog UI: http://localhost:8025
